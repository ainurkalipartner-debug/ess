# ЭлектроСтройСервис (ЭСС) — корпоративный сайт

Премиальный двуязычный (RU/KK) корпоративный сайт для энергетической инжиниринговой компании.
Чистый HTML/CSS/JS без сборки — быстрый, адаптивный, SEO-оптимизированный.

## 🚀 Запуск локально

Откройте папку `site/` в терминале и запустите один из вариантов:

**Python (рекомендуется):**
```bash
python -m http.server 5500
```
Затем откройте http://localhost:5500

**Node.js:**
```bash
npx serve .
```

**VS Code:** правый клик по `index.html` → *Open with Live Server*.

> Простое открытие `index.html` двойным кликом тоже работает, но локальный сервер
> корректнее обрабатывает шрифты, карту и изображения.

## 🎨 Дизайн-система

| Роль | Цвет |
|------|------|
| Глубокий тёмно-синий (фон) | `#070b16` / `#0b1020` / `#0f1a33` |
| Электрик-блю (акцент) | `#2d7df0` |
| Оранжевый (бренд, из логотипа) | `#ee7b1f` |
| Белый / серые | `#ffffff` … `#64748b` |

- Шрифты: **Sora** (заголовки) + **Inter** (текст).
- Анимации: scroll-reveal через IntersectionObserver, анимированные счётчики,
  hover-эффекты карточек, marquee-лента партнёров, параллакс-герой.
- Полная адаптивность: ≥1240 / 1080 / 920 / 640 px брейкпоинты.
- Учитывается `prefers-reduced-motion`.

## 🌐 Двуязычность (RU / KK)

Перевод реализован через атрибуты `data-ru` / `data-kk` (и `data-ru-ph` / `data-kk-ph`
для placeholder). Переключатель **RU / KZ** в шапке; выбор сохраняется в `localStorage`.
По умолчанию — русский.

## 🔍 SEO — что уже сделано

- Семантические теги, один `<h1>`, иерархия заголовков.
- `<title>`, `meta description`, `keywords`, `canonical`.
- Open Graph + `theme-color` для соцсетей и мессенджеров.
- `lang` + `og:locale:alternate` (ru/kk).
- Lazy-loading изображений, `alt` для всех картинок.
- Два блока **JSON-LD**: `Organization` и `WebSite`.
